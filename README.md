# ☕ Coffee Sales Dashboard (Excel)

## 📊 Project Overview

This project presents an interactive **Coffee Sales Dashboard** built using Microsoft Excel. The goal was to transform raw sales data into meaningful insights through data cleaning, analysis, and visualization.

The dashboard provides a clear view of sales performance across different dimensions such as time, country, product type, and customer segments.

---

## 🎯 Objectives

* Analyze coffee sales trends over time
* Identify top-performing countries and customers
* Understand the impact of product attributes (roast type, size, loyalty card)
* Build an interactive and user-friendly dashboard

---

## 🛠️ Tools & Techniques Used

* **Microsoft Excel**

  * Data Cleaning
  * Pivot Tables
  * Pivot Charts
  * Slicers & Timeline
  * Data Visualization

---

## 📌 Key Features

### 📅 Time-Based Analysis

* Interactive timeline to filter sales by year and month
* Line chart showing **Total Sales Over Time**
* Trend comparison across different coffee types

### ☕ Product Insights

* Sales breakdown by:

  * Roast Type (Dark, Medium, Light)
  * Coffee Type (Arabica, Excelsa, Liberica, Robusta)
  * Package Size

### 🌍 Regional Performance

* Bar chart displaying **Sales by Country**
* Identifies top-performing regions

### 👥 Customer Insights

* Highlights **Top 5 Customers** based on sales
* Helps identify high-value customers

### 🎛️ Interactivity

* Dynamic filters using slicers:

  * Roast Type
  * Size
  * Loyalty Card
* Fully interactive dashboard experience

---

## 📈 Key Insights

* The **United States** contributes the highest sales among all countries
* Certain coffee types (like Arabica & Liberica) show fluctuating but strong trends
* A small group of customers contributes significantly to revenue
* Sales patterns vary significantly over time, indicating seasonal demand
